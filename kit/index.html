The Sovereign Stack
Templates, configuration, and an operating charter. Install, configure, run.

This is the working starting point. The kit gives you everything you need to spin up a functional agent system in an afternoon.

It comes in three tiers:

Base — Templates for the core files: SOUL.md, the index, the log, the vault scaffold. Paste them into your workspace and start editing. You get the structure and the instructions to make it yours.

Professional — Adds the Build Charter and Companion Prompt. The Charter defines what the agent is allowed to do, what it must ask about, and what it must report. The Companion Prompt shapes how it thinks and responds. Plus the dashboard that tracks what’s running.

Applied — Industry-specific configurations layered on top. Different data paths, different workflows, different defaults for different work.

This site was built overnight by the agent architecture it sells. That is not a metaphor.
